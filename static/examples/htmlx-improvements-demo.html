🌙 Mishkah HTMLx - التحسينات الجديدة

تحسينات Phase 1: نظام تسمية مرن + Auto-scope + @ و : syntax

1️⃣ نظام التسمية المرن

دعم جميع أنماط التسمية:

<Modal>      JSX-style
<modal>      Vue-style
<m-modal>    Web Components
<comp-Modal> Legacy

2️⃣ Auto-Scope

لا حاجة لـ data-namespace:

<template id="my-comp">
  <style>
    /* scope تلقائي */
  </style>
</template>

3️⃣ @ و : Syntax

Syntactic sugar مثل Vue:

@click="save"
:value="data.name"
:class="className"

🎨 Demo 2: Auto-Scope Detection

CSS Scoping تلقائي بناءً على ID:

❌ الطريقة القديمة (Boilerplate)

<template
  id="my-card"
  data-namespace="my-card">

✅ الطريقة الجديدة (بسيطة)

<template id="my-card">
  <!-- namespace تلقائي -->

⚡ Demo 3: @ و : Syntax

Syntactic sugar للأحداث والربط:

❌ الطريقة القديمة (طويلة)

<button data-m-on-click="save">
<input value="{{data.name}}">

✅ الطريقة الجديدة (مختصرة)

<button @click="save">
<input :value="data.name">
